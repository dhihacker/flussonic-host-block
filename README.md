# Flussonic Host Block
Flussonic host block!  A simple script to manage local /etc/hosts entries for blocking or redirecting domains. Easily adds, removes, or replaces hostnames to 127.0.0.1 for testing, local development, or ad/tracker blocking.

Extended Details / README snippet (optional):

Supports bulk adding/removing of host entries.

Can be used to redirect multiple domains to localhost.

Useful for testing services locally without modifying DNS.

Provides ready-to-use domain blocks for development or blocking unwanted traffic.
